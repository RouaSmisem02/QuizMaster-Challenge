# Quiz Master Challenge

## Purpose of the Program
The Quiz Master Challenge is a console application that tests your knowledge in Math, Capitals of cities, and Programming. It presents questions with a 10-second timer for each question and provides immediate feedback on your answers.

## How to Run the Program
### Prerequisites
- Make sure you have .NET Core SDK installed on your machine.

### Steps to Run
1. **Clone the Repository:**
   - Clone the repository or download the source code.

2. **Navigate to Project Directory:**
   - Open a terminal or command prompt and navigate to the directory containing the `QuizMaster_Challenge.csproj` file.

3. **Run the Program:**
   - Execute the following command:
     ```
     dotnet run
     ```

4. **Instructions for Using the Program:**
   - Once the program starts, questions from Math, Capitals, and Programming will appear.
   - Each question has a 10-second timer. Enter your answer within the given time.
   - If the timer expires, the correct answer will be displayed, and the program will proceed to the next question.
   - After all questions are answered, the program will display your final score.

## Additional Information
- This program showcases basic console input/output, exception handling, and timer usage in C#.
- You can customize the quiz by editing the `questions` and `answers` arrays in the code to add more questions or change topics.
- Developed using Visual Studio or any text editor with .NET Core SDK.
